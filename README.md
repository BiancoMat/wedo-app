# WeDo App

Questa repository contiene un esempio di applicazione React sviluppata con [Vite](https://vitejs.dev/).

## Avvio dell'ambiente di sviluppo

1. Installare le dipendenze (solo la prima volta):
   ```bash
   npm install
   ```
2. Avviare il server di sviluppo esponendo la porta:
   ```bash
   npm run dev
   ```

Quando la command `npm run dev` viene eseguita, Vite avvia il server sulla porta `5173` e lo rende accessibile da remoto grazie all'opzione `--host`. In GitHub Codespaces è sufficiente aprire il link generato su `https://<nome-codespace>-5173.app.github.dev/` per vedere l'applicazione in esecuzione.
